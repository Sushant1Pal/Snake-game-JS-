# 🐍 Snake Game

A classic Snake Game built using **HTML**, **CSS**, and **JavaScript**. Control the snake, collect food, grow longer, and try to achieve the highest score without colliding with the walls or yourself.

## 🎮 Features

- Smooth snake movement
- Random food generation
- Score tracking
- Game Over detection
- Restart functionality
- Responsive and clean UI
- Pure JavaScript implementation (No external libraries)

| ➡️ Arrow Right | Move Right |

## ⚙️ How It Works

1. The snake starts at a fixed position.
2. Food appears randomly on the game board.
3. When the snake eats food:
   - Score increases.
   - Snake grows longer.
   - New food is generated.
4. The game ends if:
   - The snake hits the wall.
   - The snake collides with itself.
